# 🌍 Projeto Turismo

Um site de turismo desenvolvido com foco em apresentar destinos de viagem de forma visual e atrativa.  
O projeto foi criado para praticar conceitos de desenvolvimento front-end, incluindo estruturação com HTML, estilização com CSS e organização de layout.

---

## 🚀 Tecnologias utilizadas

- HTML5  
- CSS3  
  

---

## 🎯 Funcionalidades

- Exibição de destinos turísticos  
- Layout organizado e visual agradável  
- Estrutura pronta para expansão com novas páginas  
- (adicione mais se tiver 👇)  
- Responsividade para diferentes telas
